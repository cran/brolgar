So letâ€™s fit a model where we look at the impact of xp, unemployment rate, and fit an intercept for each individual.

library(lme4)
wages_fit_int <- lmer(ln_wages ~ xp + ged + unemploy_rate + (xp |id), 
                      data = wages)

We can use the tools from modelr to add predictions and residuals to the data

library(modelr)
wages_aug <- wages %>%
  add_predictions(wages_fit_int, var = "pred_int") %>%
  add_residuals(wages_fit_int, var = "res_int")

Now letâ€™s look at the predictions over xp

We can use keys_near() to return those specified keys that are close to these values. Because this is a tsibble object, we donâ€™t need to specify the key variable here.

wages_aug_near <- wages_aug %>%
  keys_near(var = res_int)
